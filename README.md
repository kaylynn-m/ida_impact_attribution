# ida_impact_attribution
This repository contains the code used to produce statistics, figures, and tract-level maps examining climate change attributable flooding following Hurricane Ida in south Louisiana.

## Repository Structure

scripts/        R scripts for cleaning and visualization
tract data/     Tract-level flood impact dataset and spatial data
figures/        Output figures and maps

## Key packages:
tidyverse
sf
tmap
ggplot2
cowplot

Raw parcel level data are not included in this repository.
Tract boundaries and waterbodies sourced from the US Census Bureau.
